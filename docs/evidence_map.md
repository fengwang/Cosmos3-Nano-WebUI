- Speculative claims cannot become MUST-level requirements.
- Owner decisions can become contract constraints, but technical feasibility
  still needs verification by the owning session.

  private absolute paths as *committed* facts; where a runtime mount path

  path be resolved from a configurable variable, not baked in (INV-1).

  disagree (the 720p VRAM question), the disagreement is stated and one path
  is chosen (row E-08).
